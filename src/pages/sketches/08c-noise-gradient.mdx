## noise() で有機的グラデーション

`noise()` で色相をゆらすことで、自然界のような有機的なグラデーションアニメーションを作ります。**クリックでパターンがジャンプ**します。

## 仕組み

各ピクセルの位置 `(x, y)` と時間 `t` を `noise()` に渡し、返り値を色相に変換します。

```js
p.colorMode(p.HSB, 360, 100, 100);

for (let x = 0; x < p.width; x += step) {
  for (let y = 0; y < p.height; y += step) {
    const hue = p.noise(x * noiseScale, y * noiseScale, t) * 360;
    p.fill(hue, 80, 85);
    p.rect(x, y, step, step);
  }
}

t += 0.005; // 毎フレーム少しずつ時間を進める
```

`noise()` は近い座標に対して近い値を返すため、**隣接するピクセルの色相も近く**なり、滑らかなグラデーションになります。

## noiseScale の効果

```
noiseScale = 0.002 → 大きくぼんやりした色の塊
noiseScale = 0.008 → 中程度の模様（このスケッチ）
noiseScale = 0.03  → 細かく複雑なパターン
```

`noiseScale` は座標にかける係数です。小さいほど広い範囲が同じ色になり、大きいほど変化が細かくなります。

## 複数の noise レイヤー

このスケッチでは色相だけでなく、彩度と明度にもそれぞれ別の `noise()` を使っています。

```js
// 色相
const hue = p.noise(x * 0.008, y * 0.008, t) * 360;

// 彩度（別のオフセット・スケール）
const sat = 60 + p.noise(x * 0.004 + 100, y * 0.004, t * 0.7) * 40;

// 明度（さらに別のオフセット・スケール）
const bri = 70 + p.noise(x * 0.0024 + 200, y * 0.0024, t * 0.5) * 25;
```

### レイヤー分離のコツ

| テクニック | 効果 |
|---|---|
| **オフセットを加える**（`+ 100`） | 同じ座標でも別のノイズ値になる |
| **スケールを変える**（`* 0.004`） | 変化の粗さが変わる |
| **時間係数を変える**（`t * 0.7`） | 変化速度に差が出る |

## パフォーマンスの工夫

全ピクセルを毎フレーム計算すると重くなるため、`step` を使って間引きます。

```js
const step = 4; // 4px ごとに計算
for (let x = 0; x < p.width; x += step) {
  for (let y = 0; y < p.height; y += step) {
    // ...
    p.rect(x, y, step, step); // step サイズの矩形で塗る
  }
}
```

### ポイント

1. **`noise(x, y, t)` の 3 番目の引数で時間変化** — アニメーションの鍵
2. **複数の noise レイヤーで色に深み** — H, S, B それぞれ別のノイズで制御
3. **step で間引いてパフォーマンス確保** — 見た目とのバランスで調整

## 試してみよう

- `noiseScale` を変えて、模様の細かさを調整してみましょう
- `timeSpeed` を大きくして、変化の速さを体感してみましょう
- マウス位置を `noise()` の引数に追加して、インタラクティブに模様を変えてみましょう
