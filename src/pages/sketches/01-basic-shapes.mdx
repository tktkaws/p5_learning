## 基本図形と色

p5.js はキャンバスに図形を描くためのシンプルな関数を提供しています。このスケッチでは、代表的な 4 つの図形を描画しています。

## インスタンスモードとは

このサイトでは **インスタンスモード** という書き方を使っています。
通常の p5.js チュートリアルでは `function setup()` / `function draw()` をグローバルに書きますが、インスタンスモードでは `new p5((p) => { ... })` のように書きます。

```js
// インスタンスモード
export default function (p) {
  p.setup = () => {
    p.createCanvas(400, 400);
  };
  p.draw = () => {
    p.background(240);
    p.circle(200, 200, 100);
  };
}
```

> `p.circle(...)` のように、すべての p5 関数の前に `p.` を付けるのがポイントです。

メリットは **1 ページに複数のスケッチを安全に共存** できること。Web フレームワーク（Astro など）と組み合わせるときに便利です。

## 図形の描画関数

| 関数 | 説明 | 例 |
|------|------|-----|
| `rect(x, y, w, h)` | 四角形 | `p.rect(50, 50, 120, 120)` |
| `circle(x, y, d)` | 円（d は直径） | `p.circle(200, 200, 100)` |
| `ellipse(x, y, w, h)` | 楕円 | `p.ellipse(200, 200, 160, 100)` |
| `triangle(x1,y1, x2,y2, x3,y3)` | 三角形 | `p.triangle(100,300, 50,200, 150,200)` |

## 色の指定

```js
p.fill(100, 149, 237);   // RGB で塗りの色を設定
p.noFill();               // 塗りなし
p.stroke(99, 102, 241);  // 線の色
p.noStroke();             // 線なし
p.strokeWeight(3);        // 線の太さ
```

- `fill()` はその後に描くすべての図形に適用されます
- 色の指定は `(R, G, B)` の 0〜255、または `(グレー値)` で指定できます
- `(R, G, B, A)` で透明度（0〜255）も指定可能です

## noLoop() について

このスケッチでは `p.noLoop()` を呼んでいます。これは `draw()` を 1 回だけ実行して止めるという意味です。静止画を描くときに使います。

## 試してみよう

- 色の数値を変えてみましょう（例: `p.fill(255, 200, 0)` で黄色）
- `rect()` の 5 番目の引数は角丸の半径です — `p.rect(50, 50, 120, 120, 20)` で大きく丸めてみましょう
- `p.background(0)` にすると背景が黒になります
