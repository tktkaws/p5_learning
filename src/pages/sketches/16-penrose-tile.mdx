## ペンローズタイル

非周期的なタイル貼りを描画します。**クリックでカイト＆ダート → 太い菱形＆細い菱形 → 色付きパターンに切り替え**できます。

## ペンローズタイルとは

ペンローズタイルは、**同じパターンが繰り返されない（非周期的）** のに、隙間なく平面を埋め尽くせるタイル貼りです。1974 年にロジャー・ペンローズが発見しました。

5 回対称の美しいパターンが特徴で、準結晶の構造と深い関係があります。

## ロビンソン三角形による生成

ペンローズタイルは 2 種類の三角形（ロビンソン三角形）を再帰的に分割して生成します。

| 三角形 | 頂角 | 底角 | 特徴 |
|---|---|---|---|
| 黄金三角形（type 0） | 36° | 72° | 鋭い二等辺三角形 |
| 黄金グノモン（type 1） | 108° | 36° | 鈍い二等辺三角形 |

黄金比 `φ = (1 + √5) / 2 ≈ 1.618` がすべての分割に関わります。

## 分割アルゴリズム

```js
const PHI = (1 + Math.sqrt(5)) / 2;

function subdivide(triangles) {
  const result = [];
  for (const [type, A, B, C] of triangles) {
    if (type === 0) {
      // 黄金三角形 → 1 三角形 + 1 グノモン
      const P = lerpPt(A, B, 1 / PHI);
      result.push([0, C, P, B]);
      result.push([1, P, C, A]);
    } else {
      // 黄金グノモン → 2 グノモン + 1 三角形
      const Q = lerpPt(B, A, 1 / PHI);
      const R = lerpPt(B, C, 1 / PHI);
      result.push([1, R, C, A]);
      result.push([1, Q, R, B]);
      result.push([0, R, Q, A]);
    }
  }
  return result;
}

function lerpPt(a, b, t) {
  return [a[0] + (b[0] - a[0]) * t,
          a[1] + (b[1] - a[1]) * t];
}
```

分割を繰り返すほどタイルが細かくなり、非周期パターンが明確になります。

## 初期配置

10 個の黄金三角形を中心から放射状に配置して「太陽」パターンを作り、これを分割の出発点にします。

```js
for (let i = 0; i < 10; i++) {
  const a1 = (TWO_PI / 10) * i;
  const a2 = (TWO_PI / 10) * (i + 1);
  const A = [cx, cy]; // 中心
  const B = [cx + cos(a1) * r, cy + sin(a1) * r];
  const C = [cx + cos(a2) * r, cy + sin(a2) * r];
  triangles.push([0, A, i % 2 === 0 ? B : C, i % 2 === 0 ? C : B]);
}
```

5 回分割すると約 890 個の三角形になり、ペンローズタイルの特徴的な非周期パターンが現れます。

### ポイント

1. **黄金比が鍵** — 分割点は常に `1/φ` の位置で、これが非周期性を生む
2. **2 種類の三角形** — 黄金三角形とグノモンの組み合わせですべてが構成される
3. **自己相似性** — 分割を繰り返しても同じ 2 種類の三角形しか現れない
4. **5 回対称** — 正五角形と深く関連し、72° と 36° の角度が繰り返し現れる
