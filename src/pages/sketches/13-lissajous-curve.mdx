## リサージュ曲線

`sin` と `cos` の周波数比を変えて描く美しい曲線です。**クリックで周波数比（1:2 → 2:3 → 3:4 → 3:5 → 5:6）を切り替え**できます。描画アニメーションで曲線ができる過程を観察できます。

## リサージュ曲線とは

x 軸と y 軸にそれぞれ異なる周波数の `sin` 波を当てたとき、点が描く軌跡がリサージュ曲線です。

```
x = A * sin(a * t + δ)
y = B * sin(b * t)
```

| パラメータ | 意味 |
|---|---|
| `a`, `b` | x 軸・y 軸の周波数（この比率で形が決まる） |
| `A`, `B` | x 軸・y 軸の振幅 |
| `δ` | 位相差（曲線の傾きや開き具合が変わる） |
| `t` | 時間パラメータ（0 〜 2π で 1 周期） |

## 基本の描画コード

```js
const amp = 150;
const a = 1; // x の周波数
const b = 2; // y の周波数

p.beginShape();
for (let t = 0; t < p.TWO_PI; t += 0.01) {
  const x = amp * p.sin(a * t + p.PI / 2);
  const y = amp * p.sin(b * t);
  p.vertex(p.width / 2 + x, p.height / 2 + y);
}
p.endShape();
```

位相差 `δ = π/2` を加えることで、曲線が画面の中央を通り、見栄えの良い形になります。

## 周波数比と形の関係

周波数比 `a : b` によって曲線の形が劇的に変化します。

| 周波数比 | 交差数 | 形の特徴 |
|---|---|---|
| 1 : 2 | 1 | 8 の字（最もシンプル） |
| 2 : 3 | 5 | 結び目のような複雑な曲線 |
| 3 : 4 | 11 | さらに密な交差パターン |
| 3 : 5 | 14 | 星形に近い模様 |
| 5 : 6 | 29 | 非常に複雑で繊細な模様 |

> 一般に、交差数は `2ab - a - b` で計算できます（`a` と `b` が互いに素の場合）。

## アニメーションの仕組み

一度に全体を描くのではなく、`t` を少しずつ増やして軌跡を配列に保存することで描画アニメーションを実現しています。

```js
let t = 0;
let points = [];

p.draw = () => {
  // 毎フレーム少しずつ点を追加
  if (t <= p.TWO_PI) {
    const x = amp * p.sin(a * t + p.PI / 2);
    const y = amp * p.sin(b * t);
    points.push({ x, y });
    t += 0.025;
  }

  // 保存した点を描画
  p.beginShape();
  for (const pt of points) {
    p.vertex(pt.x, pt.y);
  }
  p.endShape();
};
```

| 要素 | 役割 |
|---|---|
| `points` 配列 | 過去の軌跡を保存し、毎フレーム描き直す |
| `t += 0.025` | 1 フレームの進行量。小さいほどゆっくり描画 |
| `noLoop()` | `t > TWO_PI` で 1 周したら描画を停止 |

### ポイント

1. **周波数比が形を決める** — `a : b` の比率だけで全く異なる曲線が生まれる
2. **位相差で傾きが変わる** — `δ` を変えると曲線が回転・変形する
3. **`sin` の組み合わせ** — 2 つの単純な波の合成で複雑な模様が描ける
4. **軌跡の保存** — 配列に点を溜めて描画することでアニメーション表現が可能

## 試してみよう

- 位相差 `δ` を `0` や `π/4` に変えて形の変化を観察しましょう
- 周波数比を `1 : 1` にするとどんな形になるでしょうか？（答え: 楕円）
- 周波数を整数でない値（例: `1 : 2.01`）にすると、閉じない曲線になります
- 振幅 `A` と `B` を異なる値にして、縦長・横長のリサージュを作ってみましょう
