## HSB 色相グリッド

彩度（S）や明度（B）を段階的に変えた色相グリッドです。**クリックで 3 つのモード（H×S / H×B / 両方）を切り替え**られます。

## HSB のおさらい

```js
p.colorMode(p.HSB, 360, 100, 100);
p.fill(hue, saturation, brightness);
```

| パラメータ | 役割 | 値を下げると |
|---|---|---|
| H（色相） | 色の種類（赤→黄→緑→…） | — |
| S（彩度） | 鮮やかさ | グレーに近づく |
| B（明度） | 明るさ | 黒に近づく |

## グリッドの作り方

色相を横軸、彩度（または明度）を縦軸にして 2 重ループで描きます。

### 色相 × 彩度（明度固定）

```js
p.colorMode(p.HSB, 360, 100, 100);
for (let row = 0; row < rows; row++) {
  for (let col = 0; col < cols; col++) {
    const hue = (col / cols) * 360;
    const sat = p.map(row, 0, rows - 1, 100, 10);
    p.fill(hue, sat, 90);  // B は 90 で固定
    p.rect(col * cellW, row * cellH, cellW - 2, cellH - 2);
  }
}
```

上の行ほど彩度が高く鮮やか、下に行くほどグレーに近づきます。

### 色相 × 明度（彩度固定）

```js
const bri = p.map(row, 0, rows - 1, 100, 10);
p.fill(hue, 80, bri);  // S は 80 で固定
```

上の行ほど明るく、下に行くほど暗くなります。

## map() で値を段階的に割り当てる

```js
p.map(row, 0, rows - 1, 100, 10);
//     ↑     ↑入力範囲    ↑出力範囲
// row=0 → 100, row=7 → 10
```

`map()` はグリッド系の表現で非常によく使います。行・列番号を色や座標に変換するのに最適です。

## S と B の違いを観察する

| | S（彩度）を下げる | B（明度）を下げる |
|---|---|---|
| 向かう先 | グレー（無彩色） | 黒 |
| 色味 | 薄くなる | 暗くなる |
| 使いどころ | パステルカラー、落ち着いた配色 | 影、深み、コントラスト |

### ポイント

1. **彩度と明度は独立したパラメータ** — 同じ色相でも S と B で印象が大きく変わる
2. **`map()` でループ変数を値に変換** — グリッド描画の定番テクニック
3. **HSB モードは `push()` / `pop()` で囲む** — 他の描画に影響を与えない

## 試してみよう

- グリッドのセル数を変えて、より細かい（または粗い）色見本を作ってみましょう
- マウス位置で H の範囲を限定し、特定の色相帯だけ表示してみましょう
- 3 軸すべて（H × S × B）を使って、3D 的な色空間表示に挑戦してみましょう
