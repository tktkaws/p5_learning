## 万華鏡パターン

回転対称を使って万華鏡のような模様を自動生成するスケッチです。クリックでリセットできます。

## 座標変換の基本

p5.js では `translate()` と `rotate()` で**描画の原点と向きを変更**できます。

```js
p.translate(p.width / 2, p.height / 2); // 原点を中央に移動
p.rotate(angle);                         // 原点を中心に回転
p.ellipse(x, y, size, size);            // 回転した座標系で描画
```

> `translate` や `rotate` は**その後のすべての描画**に影響します。影響範囲を限定するには `push()` / `pop()` を使います。

## push() / pop() で状態を保存・復元

```js
for (let i = 0; i < symmetry; i++) {
  p.push();                  // 現在の変換状態を保存
  p.rotate(angle * i);       // i番目の位置に回転
  p.ellipse(x, y, size, size);

  p.push();
  p.scale(1, -1);            // Y軸を反転（鏡像）
  p.ellipse(x, y, size, size);
  p.pop();

  p.pop();                   // 変換を元に戻す
}
```

| 関数 | 役割 |
|------|------|
| `push()` | 現在の描画設定（色・変換など）をスタックに保存 |
| `pop()` | 最後に `push()` した状態に復元 |
| `translate(x, y)` | 原点を (x, y) に移動 |
| `rotate(angle)` | 原点を中心に回転（ラジアン） |
| `scale(sx, sy)` | 拡大・縮小。`scale(1, -1)` で上下反転 |

## 対称性の仕組み

```
symmetry = 8 の場合：
角度 = TWO_PI / 8 = 45°

       0°
      / \
   315°   45°
    |       |
  270° ─ ● ─ 90°
    |       |
   225°  135°
      \ /
      180°
```

8方向に同じ図形を描くことで、万華鏡のような対称パターンになります。さらに `scale(1, -1)` で鏡像を加えることで、計 16 方向の対称になります。

## HSBカラーモード

このスケッチでは `colorMode(HSB)` を使っています。

```js
p.colorMode(p.HSB, 360, 100, 100, 100);
p.fill(hue, saturation, brightness, alpha);
```

| パラメータ | 範囲 | 意味 |
|-----------|------|------|
| Hue（色相） | 0〜360 | 色の種類（0=赤, 120=緑, 240=青） |
| Saturation（彩度） | 0〜100 | 鮮やかさ（0=灰色, 100=鮮やか） |
| Brightness（明度） | 0〜100 | 明るさ（0=黒, 100=明るい） |

> 色相を時間で変化させるだけで、自動的にカラフルな表現ができます。

## 試してみよう

- `symmetry` の値を変えてみましょう（3, 6, 12 など）
- 三角関数のパラメータ `t * 1.3` や `t * 2.1` を変えると軌跡のパターンが変わります
- フェードアウトの頻度 `120` を小さくすると模様が早く消えます
